# MediViewHub

MediViewHub est un acronyme de :
"Medi" fait référence à l'univers médical, tandis que "View" signifie "visualisation" en anglais.
"Hub" évoque un centre centralisé où les utilisateurs peuvent accéder à une multitude d'images médicales stockées. Ainsi, **MediViewHub** est une plateforme (stockage et visualisation des images médicales) évoluant dans le domaine de téléradiologie et ayant pour but de centraliser le système d'imagerie médical au Sénégal sur une même plateforme, d'interconnecter les hôpitaux et enfin d'avoir une base de données large pour pourvoir utiliser les techniques d'IA.
